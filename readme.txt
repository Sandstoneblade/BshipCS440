Click Tools, then Libraries.
Click New Library and name it MSSQL.
In the Classpath tab click Add Jar/Folder and
browse and add all the jar files from the microsoft folder in the repo.
After the files are added click OK to close the dialog box.
Now right click the Libraries folder in the Project tab that is on the left
and click "add library". Search for MSSQL and add it. The jar files should show up
right under the libraries folder icon. 

The program is now ready to run.
